## New Example Programs

### 1. Control Flow
**File:** `src/examples/control-flow.js`

Demonstrates:
- Simple if statements
- if-else branches
- Nested if-else (grade calculator)
- While loop countdown
- While loop to find values

### 2. Strings
**File:** `src/examples/strings.js`

Demonstrates:
- String literals
- String concatenation with +
- Concatenating strings and numbers
- Building strings in loops
- String length with len()
- String comparison

### 3. FizzBuzz
**File:** `src/examples/fizzbuzz.js`

Classic programming challenge:
- Print 1-20
- "Fizz" for multiples of 3
- "Buzz" for multiples of 5
- "FizzBuzz" for multiples of both

### 4. Fibonacci
**File:** `src/examples/fibonacci.js`

Demonstrates:
- Iterative approach (print first 15 numbers)
- Recursive approach with function
- Watch call stack in Memory tab during animated run

### 5. Prime Numbers
**File:** `src/examples/primes.js`

Demonstrates:
- isPrime() function using modulo
- Testing specific numbers
- Finding all primes up to 50
- Counting primes up to 100
